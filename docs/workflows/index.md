# Clinical Workflows

Visual flowcharts for clinical decision-making at The Sandusky Dyslipidemia Model clinic, rendered as Mermaid diagrams.

These workflows provide a visual representation of the clinical pathways described in the [Clinical Documents]({% link clinical/index.md %}). They are intended as quick-reference aids during patient encounters.

## Diagram Color Legend

| Color | Meaning |
|:------|:--------|
| **Green** | Entry points and positive outcomes |
| **Blue** | Assessment and decision steps |
| **Orange** | Caution — requires further evaluation |
| **Red** | High-risk — urgent or aggressive action |
| **Gray** | Administrative and scheduling steps |

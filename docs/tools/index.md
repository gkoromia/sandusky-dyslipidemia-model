# Interactive Clinical Tools

JavaScript-based clinical decision support tools for use by providers at The Sandusky Dyslipidemia Model clinic. These tools run entirely in the browser — no data is transmitted or stored.

**Disclaimer:** These tools are for clinical decision support only and do not replace individualized clinical judgment. All outputs should be interpreted in the context of the individual patient.

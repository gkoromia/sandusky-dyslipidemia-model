# The Sandusky Dyslipidemia Model

A comprehensive clinical model for outpatient management of dyslipidemia based on the **2026 ACC/AHA/Multisociety Guidelines on the Management of Dyslipidemia**, augmented with advanced diagnostic and therapeutic tools.

## Overview

This repository contains the clinical documentation, workflows, patient materials, and interactive tools for The Sandusky Dyslipidemia Model — a specialized outpatient clinic serving regional patients referred for cardiovascular risk factor management with a focus on dyslipidemia.

### Key Features

- **Guideline-based framework** adhering strictly to the 2026 ACC/AHA/Multisociety Guidelines
- **Advanced Tools** beyond standard guidelines: ApoB, Labcorp NMR LipoProfile, Lipoprotein(a), Coronary Artery Calcium scoring
- **"Lower is better" philosophy** using advanced biomarkers to drive aggressive lipid control
- **Interactive clinical tools** including the AHA PREVENT risk calculator
- **Provider-facing protocols** with numbered sections and citations for easy reference
- **Patient education materials** written at a 6th–8th grade reading level

### Repository Structure

```
docs/
├── clinical/       # Provider-facing clinical documents (01–12)
├── workflows/      # Mermaid-based visual flowcharts
├── tools/          # Interactive JavaScript decision support tools
└── patient/        # Patient-facing educational materials
```

## Deployment

This site is built with Jekyll using the [Just the Docs](https://just-the-docs.com/) theme and deployed via Cloudflare Pages.
